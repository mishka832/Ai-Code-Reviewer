## Overview

This project serves as an AI-powered assistant for developers to receive automated feedback on their code. It provides clear explanations and visual insights to help improve code quality and maintainability.

---

## Features

- AI-based automated code review using OpenAI API  
- Real-time response and feedback  
- Clean and responsive user interface built with Tailwind CSS  
- Visual representation of results using Recharts  
- Secure backend integration with Node.js and Express.js  

---

## Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**AI Integration:** OpenAI API  
**Data Visualization:** Recharts  

---

## How It Works

1. The user enters or pastes code into the editor.  
2. The app sends the code to the backend server.  
3. The backend communicates with the OpenAI API to analyze the code.  
4. The API returns suggestions and analysis.  
5. The results are displayed clearly on the frontend, with visual analytics.
